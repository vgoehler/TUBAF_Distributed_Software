# TUBAF_Distributed_Software

Slides for the Course Distributed Software of TUBAF

## Course Overview

This repository contains the course materials for the Distributed Software course at TU Bergakademie Freiberg.

- [Lesson 03: Verteilte Software Charakteristika und Patterns](vorlesung_03_architektur.md)
- [Lesson 04: Verteilte Software Architektur Blockaufgabe](vorlesung_04_architektur.md)

## Related Courses

This course is part of the Distributed Software Curriculum, which includes:

- [Introduction to n8n](https://github.com/vgoehler/introduction-to-n8n) - An introduction to n8n, a workflow automation platform for distributed systems

## LiaScript

All courses are written with LiaScript. LiaScript is a markup language for creating interactive and engaging educational content. It allows for the integration of multimedia elements, quizzes, and interactive exercises to enhance the learning experience.
Style information are hosted in an external repository https://raw.githubusercontent.com/vgoehler/LiaScript_CSS_Provider .
